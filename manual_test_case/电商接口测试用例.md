用户模块 - POST 电商登录

请求地址：{{baseUrl}}/user/login

请求方式：POST

请求体JSON：

{

    "username":"test01",
    
    "password":"123456"

}

前置：无

后置脚本：提取token存入环境变量

断言：

1. 响应码等于200

2. 返回data包含token字段


商品模块 - GET 商品列表

请求地址：{{baseUrl}}/goods/list?page=1&size=10

请求方式：GET

Query参数：page=1，size=10

鉴权头：Authorization: Bearer {{token}}

断言：

1. 状态码200

2. 返回存在data字段

3. data为数组


订单模块 - POST 创建订单

请求地址：{{baseUrl}}/order/create

请求方式：POST

请求体携带商品ID

鉴权头：Bearer {{token}}

后置脚本：提取orderId存入环境

断言：

1. 状态码200

2. 返回存在orderId


订单模块 - PUT 取消订单

请求地址：{{baseUrl}}/order/cancel

请求方式：PUT

传参：订单id={{orderId}}

鉴权头：Bearer {{token}}

断言：
1. 状态码200
2. 订单状态为已取消
